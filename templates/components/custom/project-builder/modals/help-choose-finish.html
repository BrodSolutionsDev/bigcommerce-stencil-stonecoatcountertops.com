What color system are you using?

  • If you are using a color system we offer, select the system you want
  • Otherwise, select custom and then choose your colors

Choose your final finish

  • If you want your finish to appear shiny, then you'll select "Ultimate Top Coat Gloss Finish"
  • If you want your finish to appear softer, then you'll select "Ultimate Top Coat Matte Finish"
  • If you don't want any protective surface layer, then you'll select "No Protective Surface Needed"
