WHAT IS YOUR PROJECT SURFACE?

  • If you are starting a new project over bare wood, then you'll select "New Countertop"
  • If this is a remodel of a countertop, select "Existing Countertop"
    *Remodel projects come with Bonding Primer

WHAT UNDERCOAT DO YOU NEED?

Like a tinted primer when painting walls, match the undercoat tone to your project pour for optimal bonding and color results.
  • If the final color will be dark, then you'll select "Black"
  • If the final color will be light, then you'll select "White"
    *White countertops come with Art Coat for UV protection against yellowing
  • If the final color will be grey (or similar), then you'll select "Grey"
    *Grey comes with both Black and White undercoats that you'll mix together
